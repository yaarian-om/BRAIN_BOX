General steps for feature extraction from an image¶

  • Pre-processing

    Before feature extraction, it is important to pre-process the image to improve its quality and remove any unnecessary information. This may include cropping, resizing, and normalization.

  • Feature selection

    There are different types of features that can be extracted from an image such as color, texture, shape, and edges. The choice of features depends on the specific problem at hand. For instance,shapes may be important, while in image classification, texture may be more relevant

  • Feature extraction

    Once the features have been selected, they can be extracted from the image using techniques such as histogram of oriented gradients (HOG)

  • Feature representation

    Feature representation: The extracted features may need to be further represented in a more compact form for efficient processing. This can be achieved through techniques such as Histogram.

  • Classification

    Finally, the extracted and represented features can be used to classify or recognize the image using machine learning algorithms such as support vector machines (SVM), decision trees, or neural networks

Feature Extraction(HOG Shape)¶

HOG Shape :¶

  • Compute the gradient magnitude and orientation at each pixel in the image.
  • Group the pixels into cells (small, overlapping regions of the image).
  • Construct a histogram of gradient orientations for each cell.
  • Normalize the histograms for each cell by the sum of the magnitudes of the gradients in the cell
  • Concatenate the histograms for all cells in the image to form the final feature vector
